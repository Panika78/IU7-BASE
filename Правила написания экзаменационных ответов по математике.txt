Правила написания экзаменационных ответов по математике

Общая задача

Ты получаешь текст конспекта по математике и должен привести его в экзаменационный формат LaTeX (вопрос–ответ).

Ты оформляешь только один раздел, так чтобы его можно было напрямую вставить в уже существующий .tex файл.

Запрещено:

- менять формулировки
- сокращать текст
- добавлять объяснения
- переставлять части текста
- заменять обозначения
- исправлять стиль автора
- дополнять материал своими знаниями

Разрешено только:

- структурировать текст
- оформлять формулы в LaTeX
- приводить оформление к единому виду

Допустимо исправлять очевидные ошибки, но их необходимо указать после кода.

Структура раздела

1. Заголовок раздела

\section{Название раздела}

2. Вопросы

Каждый вопрос оформляется как:

\subsection{Название вопроса}

\examquestion{Текст вопроса}

Разрешено использование \subsubsection{} при необходимости.

Ответы

1. Обычный ответ

\examanswer{
...
}

2. Теорема / закон / ключевое утверждение

\examtheorem{
...
}

Слово "Ответ" в этом случае не используется.

3. Доказательство

\examproof{
...
}

Сложный ответ (ключевое правило)

Если ответ содержит несколько сущностей (например, определения, теоремы и т.д.), то:

- \examanswer{} можно не использовать
- либо использовать только для вводного текста
- определения и теоремы выносятся в отдельные блоки

Определения

Все определения оформляются отдельным блоком:

\examdefinition{
...
}

Требования:

- ключевое слово выделяется жирным через \term{}
- важные слова выделяются курсивом через \important{}

Пример:

\examdefinition{
\term{Предел функции} называется число $A$, если ...
}

Если определение имеет номер, он сохраняется дословно:

\term{Опр. 3.1 (слово)} ...

Теоремы

Все теоремы оформляются строго через:

\examtheorem{
...
}

Сюда относятся:

- теоремы
- законы
- фундаментальные утверждения
- основные формулы

Формулы

Все формулы оформляются в LaTeX:

\[
...
\]

Разрешено:

- объединять формулы для читаемости
- использовать стрелки следствия

Запрещено:

- менять обозначения
- упрощать выражения
- вводить новые обозначения

Векторные обозначения

Используются строго:

\vec a, \vec b

Скалярное произведение:

(\vec a, \vec b)

Порядок текста

Порядок должен полностью совпадать с исходным.

Запрещено:

- переставлять абзацы
- менять порядок доказательства
- добавлять пояснения
- сокращать текст

Ошибки в исходном тексте

Если обнаружена ошибка:

- исправлять её в тексте запрещено
- текст сохраняется дословно
- после LaTeX-кода выводится список ошибок

Стиль оформления

- не добавлять комментарии
- не добавлять пояснения
- не использовать разговорный текст

Итоговый формат ответа

Нейросеть должна выводить:

- только LaTeX-код
- без пояснений вне кода

Код должен быть полностью готов к вставке в .tex файл без редактирования.
